## BookPop: Exchange your books with other Bookies!

This is a personal open-source project with two main different purposes:

### 1- Practice full-stack developing

I'm currently really interested on full-stack developing, from the process of creating your very first directory to a professional deploy with all set-on.
I have used different professional technologies for BookPop, some of which are:

- Next.JS
- PostgreSQL
- TypeScript

Also, I have implemented various real-life processes, such as:

- Signup, Login and Logout with password-hashing
- Authentication and Authorization using session tokens
- And more

Any feedback would be appreciated. I'm currently learning daily to make this project and my skills even better. Don't hesitate on reaching me out!

## 2- Create a real life book community

I got inspired by a really regular book reading friend for making this project. He got a problem that a lot of us that read have but never find a solution:

I have a lot of old books and want no get new ones but I don't know if I'll buy something that later I won't read because of possible many reasons:
maybe I don't like it, maybe it is not what I expected, and so on.

What should I do? Sure, I can read the pdf preview online, but it is uncomfortable, hard to find and a physical book will always be better.
Maybe a classical book would be available in a library, right? Well, maybe, but a big part are really hard to find, and you only have one option: to buy them.

So, what is BookPop? BookPop is an online community that lets you exchange books for other ones you want to read but can't afford with other local users.
You can exchange books permanently, temporarily, lend a book in exchange of a fee... The choice is yours!

BookPop is currently on a developing phase, and it will be first only available in Barcelona, Spain.

If you want all info about it, you can reach me!
